Detailed Deliverables by Module

Concrete technical outcomes and implementation pillars

Module 1

Base Package: Salesforce Live Chat

  • Azure Instance Provisioning: Setup and configuration of Microsoft Azure App Service for .NET orchestration middleware.

  • Security & Networking: Configuration of Azure Key Vault and implementation of VNet for Salesforce connectivity.

  • React UI Framework: Development of a custom, responsive chat window to replace legacy Salesforce Live Agent UI.

  • Human Escalation: Configuration of "Human-in-the-loop" handshake routing to the Salesforce Service Console.

  • State Management: Implementation of persistent chat history for the duration of the user session.

Module 2

Multilingual Add-on (Microsoft Translator)

  • Real-time API Integration: Direct connection to Microsoft Translator API for high-speed, on-the-fly translation.

  • Language Support: Full localization support for Farsi (RTL) and Malaysian.

  • Translation Toggle: UI elements allowing both users and agents to view original and translated text simultaneously.

Module 3

AI Knowledgebase (Bezaka AI)

  • Bezaka Engine Setup: Integration of the proprietary Bezaka.AI engine to serve as the primary response layer.

  • Context Retrieval: Logic to pull account history from Salesforce for personalized, context-aware AI answers.

  • Escalation Triggers: Automated hand-off to Module 1 when AI confidence falls below the set threshold.

Project Phasing & Timeline

The project is structured into three incremental modules over a 12-week period. Windsor Brokers may elect to proceed with subsequent modules upon successful completion of the preceding phase.
